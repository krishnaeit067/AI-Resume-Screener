# AI Resume Screener

An intelligent AI-powered Resume Screening and Candidate Evaluation System built using **Python** and the **OpenAI API**. The application analyzes uploaded resumes, extracts relevant information, evaluates candidate profiles, and provides AI-generated hiring recommendations.

---

## Features

- Upload resumes in PDF format
- Automatic resume text extraction
- AI-powered candidate evaluation
- Skill extraction and categorization
- Resume summarization
- ATS-style resume scoring
- Hiring recommendation
- User-friendly interface
- Fast and accurate resume analysis

---

## Tech Stack

- Python
- OpenAI API
- Streamlit
- PyPDF2
- Pandas
- NumPy
- python-dotenv

---

## How It Works

1. Upload a resume.
2. Extract text from the PDF.
3. Send resume content to the OpenAI API.
4. Analyze:
   - Skills
   - Experience
   - Education
   - Strengths
   - Weaknesses
   - ATS Compatibility
5. Generate a hiring recommendation and detailed analysis.

## Installation

```bash
git clone https://github.com/yourusername/AI-Resume-Screener.git

cd AI-Resume-Screener

pip install -r requirements.txt
```

Create a `.env` file:

```env
OPENAI_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
streamlit run app.py
```
